- **Создание собственного меню.**
- **Создание своего автономного сервера развертки образов.**
****
## Используемый стек:
- **Alterator-web.**
>	Автоматическая подготовка требуемых initrd.img и vmlinuz.
- **Dnsmasq.**
>	Автоматическая выдача IP-адресов при загрузке через PXE, TFTP сервер.
- **Nfs-сервер.**
>	Получение образа по сети.
- **Http-сервер (nginx)**
>	Получение файла ответов по сети.
****

| Сервис       | Протокол     | Проверка    |
| ------------ | ------------ | ----------- |
| dnsmasq DHCP | UDP 67       | `ss -ulpn`  |
| dnsmasq TFTP | UDP 69       | `ss -ulpn`  |
| nginx        | TCP 80       | `ss -tulpn` |
| NFS          | TCP/UDP 2049 | `ss -tulpn` |

****
## Настройка dnsmasq
```bash
sudo nano /etc/dnsmasq.d/pxe-server.conf
```
- Сам конфиг файл такой:
```bash
interface=enp0s8
bind-interfaces

# DHCP
dhcp-range=192.168.56.100,192.168.56.200

# PXE
dhcp-boot=pxelinux.0

# TFTP
enable-tftp
tftp-root=/srv/public/netinst

# LOGS
log-dhcp
log-queries
```
****
## Создание каталогов для развертки образов.
- Создание каталога для образов
```bash
sudo mkdir -p /srv/iso
```

- Создание каталога для файлов конфигурации
```bash
sudo mkdir -p /srv/pxe
```

- Создание каталогов для tftp сервера и образов initrd и vmlinuz
```bash
cd /srv/pxe
sudo mkdir -p images pxelinux.cfg
```

- Создание каталогов для развертки нескольких операционных систем
```bash
cd images
sudo mkdir -p kworkstation workstation server
```

**После создания каталогов нужно в веб версию альтератора и импортировать образ**
