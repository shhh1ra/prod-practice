## Синтаксис команды
```bash
ss -tulpn
```
- **-t** - TCP
- **-u** - UPD
- **-l -** только слушающие сокеты
- **-p** - показать процесс владельца
- **-n** - не резолвить имена
****
## Разница между -tulpn и -ulpn
```bash
ss -tulpn
```
- Показывает все TCP и UDP сервисы одновременно, примерный вывод:
```
Netid State  Local Address:Port
tcp   LISTEN 0.0.0.0:80
tcp   LISTEN 0.0.0.0:22
udp   LISTEN 0.0.0.0:69
```

```bash
ss -ulpn
```
- Показывает UDP сервисы, примерный вывод:
```
udp  UNCONN 0.0.0.0:69
udp  UNCONN 0.0.0.0:67
```
