# Отчёт по развёртыванию ALT Workstation 11.1 по сети

## Цель работы

Настроить инфраструктуру сетевой установки ALT Workstation 11.1 с использованием PXE и автоматической установки.

## Исходная инфраструктура

Использовалась среда VirtualBox:
- ALT Server — сервер развёртывания;
- ALT Workstation — клиентская машина.

Сетевая схема:
- NAT — доступ сервера в интернет;
- Host-Only — сеть развёртывания.

Адрес сервера: 192.168.56.10/24

## Ход выполнения работы

### Подготовка сервера
Настроен второй сетевой интерфейс, DHCP через dnsmasq и среда PXE.

### Настройка PXE
Через Alterator импортирован образ ALT Workstation 11.1, сформированы PXE-конфигурации (alt workstation initrd.img и vmlinuz) и каталог /srv/public/netinst.

### Проверка загрузки клиента
Проверена выдача IP, загрузка PXE-загрузчика, ядра и initrd.

### Настройка NFS
Настроен экспорт каталога /srv/public для доступа установщика к образу.

### Настройка Autoinstall
Создан и опубликован файл autoinstall.scm через nginx в каталоге /var/www/html/metadata.

### Автоматическая установка
Настроены локаль, сеть, разметка диска, пользователь, GRUB и установка базовой системы.

## Возникшие проблемы
- Настройка NFS для доступа к образу.
- Отсутствие HTTP-сервера для metadata.
- Ошибки в autoinstall.scm.
- Проблемы с установкой GRUB.

## Результат
Реализована автоматическая установка ALT Workstation 11.1 по сети с использованием DHCP, PXE, TFTP, NFS, HTTP и Autoinstall.
